Materi Raspberry Pi: 22 Topik untuk Satu Semester

Narasi Script:

"Ringkasan: Materi pengenalan ini penting sebagai fondasi, di mana kita mengenal Raspberry Pi sebagai single-board computer (SBC) yang murah dan powerful, dibuat oleh Raspberry Pi Foundation di Inggris. Komponen utamanya termasuk prosesor ARM, RAM hingga 8GB, port USB, HDMI, GPIO pins, dan slot microSD. Berguna untuk proyek IoT, robotik, server rumahan, atau bahkan media center karena konsumsi daya rendah (sekitar 5-7W) dan harga terjangkau di bawah Rp1 juta. Ini juga mendukung berbagai OS seperti Raspberry Pi OS berbasis Linux, Ubuntu, atau Windows IoT.

Cara: Mulai dengan unboxing Raspberry Pi model 4 atau 5 – pastikan ada heatsink jika overclock. Download Raspberry Pi OS dari situs resmi raspberrypi.com/downloads, gunakan tool Raspberry Pi Imager untuk flash ke microSD card minimal 16GB Class 10. Colok power supply 5V 3A resmi (jangan pakai charger HP biasa biar stabil), hubungkan HDMI ke monitor/TV, dan keyboard-mouse USB. Boot pertama kali akan setup wizard sederhana.

Hasil: Raspberry Pi booting ke desktop Raspberry Pi OS dalam waktu kurang dari 1 menit, tampilan mirip Ubuntu dengan menu start, file explorer, dan browser Chromium siap browsing. Kalian bisa coba buka terminal dan ketik 'uname -a' untuk konfirmasi versi kernel Linux."

(Demo: Tampilkan proses unboxing zoom-in, flashing SD card di PC, booting sequence, dan eksplorasi desktop dengan mouse cursor bergerak. Tampilkan spec hardware via command 'vcgencmd get_config int'.)

Narasi Script:

"Ringkasan: Linux adalah OS utama Raspberry Pi, dan dasar CLI (Command Line Interface) ini krusial karena lebih efisien untuk scripting dan troubleshooting daripada GUI. Perintah seperti ls (list files), cd (change directory), mkdir (make directory), nano (text editor sederhana), dan sudo (superuser do) membantu navigasi filesystem, edit file, dan jalankan command dengan hak akses tinggi. Ini berguna untuk automation, seperti cron jobs, dan memahami konsep Unix philosophy 'do one thing well'.

Cara: Buka terminal dari ikon di taskbar atau Ctrl+Alt+T. Coba navigasi ke home directory (~), buat folder baru, edit file teks. Gunakan man [perintah] untuk baca manual jika bingung.

Hasil: Terminal menampilkan daftar file/folder dengan warna berbeda (biru untuk directory), atau file teks baru yang dibuat muncul di file manager GUI, membuktikan CLI dan GUI terintegrasi."

(Demo: Screen terminal dengan output perintah dasar seperti 'ls -la' showing hidden files, tanpa ketik detail code. Zoom-in pada output untuk jelaskan arti kolom seperti permissions dan size.)

Narasi Script:

"Ringkasan: Manajemen file/folder melatih organisasi data, sementara perbedaan root (superuser) vs user biasa penting untuk keamanan – root punya akses penuh ke sistem (/etc, /bin), user biasa terbatas ke home directory untuk hindari kerusakan accidental. Ini dasar konsep permission di Linux (read/write/execute) yang dicek via ls -l. Berguna untuk multi-user setup atau server.

Cara: Gunakan file manager (pcmanfm) untuk drag-drop copy/move file, atau terminal dengan cp/mv/rm. Untuk root, gunakan sudo sebelum command sensitif.

Hasil: File berhasil dipindah ke folder baru tanpa error, tapi kalau coba edit file sistem sebagai user biasa, muncul 'permission denied'; switch ke root via sudo su, lalu berhasil – tapi ingat, gunakan root hati-hati!"

(Demo: Before-after file structure di explorer, tampilkan error message untuk user biasa dan sukses untuk root. Gunakan split-screen untuk bandingkan.)

Narasi Script:

"Ringkasan: Konfigurasi jaringan ini esensial untuk koneksi stabil, hostname untuk identifikasi mudah di network (misal myraspi.local), dan IP static agar alamat tak berubah saat restart – berguna untuk server atau akses remote tanpa DHCP issues. Ini juga perkenalan dengan networking basics seperti DHCP vs static.

Cara: Dari taskbar, klik ikon WiFi untuk scan dan connect ke SSID, masukkan password. Ubah hostname via menu Preferences > Raspberry Pi Configuration. Untuk IP static, edit /etc/dhcpcd.conf via nano (pakai sudo). Restart network setelahnya.

Hasil: Raspberry Pi terkoneksi WiFi dengan signal strength bagus (cek via iwconfig), hostname baru terdeteksi di router admin page, dan IP tetap sama (misal 192.168.1.100) meski reboot – tes dengan ping dari laptop."

(Demo: Ping sukses dari laptop command prompt, tampilkan IP via 'ifconfig' atau 'ip addr', dan router page screenshot.)

Narasi Script:

"Ringkasan: SSH (Secure Shell) untuk akses terminal remote aman via enkripsi, VNC (Virtual Network Computing) untuk full desktop remote – berguna saat Raspberry Pi headless (tanpa monitor), seperti di robot atau sensor remote. Ini hemat ruang dan energi.

Cara: Enable SSH dan VNC via sudo raspi-config > Interfacing Options. Download RealVNC Viewer gratis dari realvnc.com di laptop/PC (support Windows/Mac/Linux). Connect via IP Raspberry Pi dan password default 'raspberry' (ubah segera!).

Hasil: Dari laptop, SSH buka terminal Raspberry Pi (gunakan PuTTY di Windows), atau VNC tampilkan full desktop Raspberry Pi – coba kontrol mouse dan ketik command dari jauh, latency rendah di LAN."

(Demo: Split-screen laptop dan Raspberry Pi, kontrol real-time seperti buka file dari VNC.)

Narasi Script:

"Ringkasan: SCP (Secure Copy) adalah extension dari SSH untuk transfer file aman, lebih cepat dan aman daripada USB atau email – ideal untuk upload script atau download log dari remote. Ini gunakan protokol SFTP underneath.

Cara: Dari terminal laptop (gunakan Git Bash di Windows), ketik scp file.txt pi@ip_raspi:/home/pi/. Untuk download sebaliknya. Pastikan SSH enabled.

Hasil: File dari laptop muncul di folder Raspberry Pi dalam detik, ukuran dan timestamp sama – tes buka file di Raspberry Pi untuk verifikasi isi."

(Demo: Before-after file di kedua device via ls, transfer progress bar jika visible.)

Narasi Script:

"Ringkasan: Python 3 adalah bahasa scripting utama di Raspberry Pi karena mudah dibaca, library豊富 seperti RPi.GPIO, dan cocok untuk beginner sampai AI. Kita bahas variabel (int, str, list), kondisi (if-else), perulangan (for/while) – berguna untuk automation sensor nanti.

Cara: Buka Thonny IDE bawaan dari menu Programming, buat file .py baru, save dan run. Install Python jika perlu via apt (tapi biasanya sudah ada).

Hasil: Skrip berjalan di Thonny console, misal variabel nama='Helmy' di-print, loop for cetak angka 1-10, atau if kondisi cek bilangan genap/ganjil dengan output benar."

(Demo: Output skrip di console Thonny, tanpa tampil code full – zoom pada hasil print statements.)

Narasi Script:

"Ringkasan: GPIO (General Purpose Input/Output) adalah 40 pin di Raspberry Pi untuk interfacing hardware – input baca sinyal (button), output kirim (LED). Ini dasar embedded system, paham pull-up/down resistor untuk stabilitas.

Cara: Hubungkan LED ke pin GPIO 17 (output) dengan resistor 330 ohm ke ground – wiring: anode LED ke GPIO, katode ke resistor lalu GND. Jalankan skrip Python dengan library gpiozero atau RPi.GPIO.

Hasil: LED nyala-mati bergantian setiap detik, simulasi blink seperti heartbeat – tambah button input untuk toggle LED saat ditekan."

(Demo: Wiring diagram animasi, LED blinking real-time dengan close-up camera.)

Narasi Script:

"Ringkasan: Sensor ini perkenalan interfacing real-world: HC-SR04 ultrasonik ukur jarak via echo sound (0-400cm), LM35 sensor suhu analog ( -55°C to 150°C). Berguna untuk robot avoidance atau monitoring lingkungan.

Cara: Wiring ultrasonik: VCC ke 5V, GND ke GND, Trigger ke GPIO 23, Echo ke GPIO 24 (pakai resistor divider jika perlu). LM35: Vout ke ADC seperti MCP3008 channel 0, karena Raspberry Pi no built-in analog. Download library gpiozero atau adafruit-circuitpython dari PyPI via pip. Beli sensor di Tokopedia/Shopee.

Hasil: Ultrasonik deteksi jarak objek (misal tangan) dan print 'Jarak: 15cm', LM35 tampilkan suhu ruangan '25.5°C' – demo dengan gerak objek mendekat."

(Demo: Sensor beraksi dengan nilai berubah di terminal, grafik sederhana jika pakai matplotlib tapi tanpa code.)

Narasi Script:

"Ringkasan: Visualisasi data real-time di terminal untuk debugging cepat, bisa tambah timestamp atau format JSON – berguna sebelum kirim ke cloud.

Cara: Hubungkan sensor seperti materi 9, jalankan skrip loop untuk baca dan print data setiap 1 detik.

Hasil: Stream nilai sensor muncul terus-menerus di terminal, misal 'Waktu: 10:00 - Suhu: 26°C - Jarak: 20cm', update live saat kondisi berubah."

(Demo: Scrolling console dengan data update, highlight baris baru.)

Narasi Script:

"Ringkasan: UART (Universal Asynchronous Receiver-Transmitter) untuk serial comm sederhana (TX/RX), I2C (Inter-Integrated Circuit) untuk multi-device bus (SDA/SCL) – berguna hubungkan modul seperti GPS atau display OLED tanpa banyak pin.

Cara: Enable UART/I2C via sudo raspi-config > Interfacing Options > Serial/I2C. Wiring: UART TX ke RX device, I2C SDA ke SDA. Tes dengan modul sederhana seperti Arduino sebagai slave.

Hasil: Data dari device eksternal diterima Raspberry Pi, misal string 'Hello' via UART muncul di minicom terminal, atau I2C scan deteksi address device tanpa error."

(Demo: Komunikasi sukses dengan lampu indikator nyala, output di serial monitor.)

Narasi Script:

"Ringkasan: BH1750 adalah sensor cahaya digital via I2C (0-65535 lux), akurat untuk auto-brightness atau plant monitoring – hemat pin karena bus sharing.

Cara: Wiring: VCC 3.3V, GND, SDA GPIO2, SCL GPIO3. Install library smbus atau adafruit-bh1750 via pip.

Hasil: Nilai intensitas cahaya ditampilkan, misal 'Lux: 100' di ruang gelap, naik ke 'Lux: 5000' saat flashlight menyinari – demo variasi cahaya."

(Demo: Sensor respon real-time dengan nilai berubah, chart sederhana.)

Hasil: Data sensor seperti suhu muncul di dashboard Thingspeak sebagai grafik line chart, update setiap 15 detik – akses dari HP browser."

Narasi Script:

"Ringkasan: MQTT (Message Queuing Telemetry Transport) adalah protokol lightweight untuk IoT, model pub-sub: publisher kirim data ke topic di broker, subscriber listen – efisien untuk low-bandwidth, seperti sensor battery-powered.

Cara: Gambar diagram pub-sub dengan tools seperti draw.io: client1 pub ke 'sensor/suhu', client2 sub dan terima.

Hasil: Pesan dari publisher (misal 'Suhu: 25') diterima subscriber secara real-time, demo dengan dua terminal simulasi."

(Demo: Animasi pesan terkirim via arrows, console output sync.)

Narasi Script:

"Ringkasan: Mosquitto adalah MQTT broker open-source, setup di Raspberry Pi jadi server lokal untuk jaringan IoT private – hindari latency cloud.

Cara: Install via sudo apt install mosquitto mosquitto-clients, start service dengan systemctl. Konfig di /etc/mosquitto.conf jika perlu port custom.

Hasil: Broker berjalan (cek sudo systemctl status mosquitto), client test pub-sub lokal sukses tanpa error, pesan terkirim antar terminal."

(Demo: Status active di terminal, test command mosquitto_pub dan _sub.)

Narasi Script:

"Ringkasan: Keamanan dasar cegah hack seperti Mirai botnet: ubah password default, enable firewall, disable root SSH, gunakan TLS untuk MQTT enkripsi – penting untuk device exposed ke internet.

Cara: Ubah password via passwd, install ufw firewall (sudo apt install ufw), edit sshd_config untuk disable root. Untuk MQTTS, generate cert dengan openssl.

Hasil: Tes SSH login root gagal, firewall block port tak diizinkan, MQTT koneksi aman dengan padlock icon di client – demo serangan simulasi ditolak."

(Demo: Error messages untuk akses unauthorized, secure connection log.)
